Un **paradigma de programación** es un estilo, enfoque o manera de pensar para estructurar y organizar el código de un programa. No es un lenguaje de programación en sí, sino una "filosofía" o un conjunto de reglas sobre cómo resolver problemas mediante código.

**Ejemplo**
Imagina que tienes que construir una casa. Puedes abordarlo construyendo todo con ladrillos y cemento (un paradigma), usando una estructura de madera (otro paradigma) o ensamblando módulos prefabricados (otro más). Cada método tiene sus propias reglas y herramientas, aunque el objetivo final (la casa) sea el mismo.
